Form Group

A Reactive Form Group is a good way to build forms and with Angulars built in FormsModel and ReactiveFormsModel it is easy to build great a schema that translates into a JSON message that can be fetched by an API to perform a CRUD operation with that data. Using the built in Angular forms module, you can see when the form changes in the input box it also changes in the JSON output. This is taking advantage of the two-way data binding Angular has to offer. Databases can easily recieive JSON data and this acts as a great way to begin shapping the data that your website needs to function. With a JSON based database however it is much easier to manage as the data becomes more complex because GraphQL and databases like FaunaDB can help build universal GraphQL queries and mutations to perform CRUD operations on various cloud and local database options.
