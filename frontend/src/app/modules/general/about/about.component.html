About


This is an Angular 11 CRUD Demo


The original code is from https://github.com/ganatan/angular11-app. This application is designed extremely well I wanted to take the existing functionality and bring it to its full potential. I have worked with dashboards, data models, data and server-based backends and web front-ends for a while and have been taking these skills and applying them primarily in the JavaScript world on both the front and backend. This applications goes about using well-known techniques for front-end design as well as for generating dynamic components using the template driven aspects of Angular to generate front-end components based on data recieved from a back-end.
